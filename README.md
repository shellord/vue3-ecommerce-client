# vue3-ecommerce-client
A vue3 ecommerce webapp

## Backend
[Backend Repository](https://github.com/shellord/vue3-ecommerce-server)

## TechStack
 - vue3
 - tailwindcss
 - apollo
 - graphql
 
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
